# 働き方タイプ診断

新卒採用候補者向けの「働き方タイプ診断」Webコンテンツです。
7問の質問に回答すると、4つの働き方タイプのいずれかが診断結果として表示されます。
自分がどんな環境で力を発揮しやすいのか、どんなチームや仕事の進め方と相性が良いのかを知ることができます。

## できること

- 7問の質問への回答でタイプ診断
- 結果として「タイプ名／キャッチコピー／強み／向いている働き方／環境／活躍場面／アドバイス」を表示
- 同点の場合は複合タイプとして両方表示

## 触り方（テストする人向け）

1. このフォルダを ZIP のまま受け取った場合は展開する
2. フォルダの中の `index.html` をダブルクリック
3. ブラウザが開いて START 画面が出る
4. 「START」ボタンから診断開始

## 4タイプ

- コツコツ集中タイプ
- チーム巻き込みタイプ
- アイデア挑戦タイプ
- 課題発見・改善タイプ

## 使用技術

- HTML / CSS / JavaScript（フレームワーク・外部ライブラリなし）
- AI API は未使用。結果テキストは事前に用意したものを表示しています
- ローディング画面は「分析中」の演出のため、意図的に1秒待っています
